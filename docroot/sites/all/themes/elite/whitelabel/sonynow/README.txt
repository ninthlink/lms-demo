This directory contains images and styles for the Sony Now whitelabel theme.

ENABLING
--------
1. In the theme settings (/admin/appearance/settings/elite), click the 'Toggle styles'
tab, then enable the 'Sony Now styles' stylesheet.
2. Also in the theme settings, set the Logo path to the following:
sites/all/themes/elite/whitelabel/sonynow/logo.png
3. Disable the 'Footer Copyright' block?
