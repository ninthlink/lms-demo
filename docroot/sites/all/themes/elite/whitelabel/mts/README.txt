This directory contains images and styles for the MTS whitelabel theme.

ENABLING
--------
1. In the theme settings (/admin/appearance/settings/elite), click the 'Toggle styles'
tab, then enable the 'MTS Whitelabel styles' stylesheet.
2. Also in the theme settings, set the Logo path to the following:
sites/all/themes/elite/whitelabel/mts/logo.png
3. Disable the 'Footer Copyright' block.
